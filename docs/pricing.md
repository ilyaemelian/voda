# Pricing

VODA is a recurring subscription product.

## Russia

| Plan | Price | Positioning |
| --- | ---: | --- |
| Start | 1,990 RUB/month | calendar, reminders, basic matching |
| Path | 2,990 RUB/month | personalized route, priorities, achievement tracking |
| Pro | 3,990 RUB/month | strategy, AI guidance, parent reports |

Main plan: **Path**.

## United States

| Plan | Price | Positioning |
| --- | ---: | --- |
| Start | $19/month | opportunity discovery and reminders |
| Path | $39/month | personalized competition/admissions route |
| Pro | $79/month | premium profile strategy and guidance |

Main plan: **Path**.

## Conversion Model

VODA should start with a free personalized preview:

1. Student or parent completes a short profile.
2. VODA returns 3 matched opportunities.
3. Parent receives a clear explanation of why they matter.
4. User upgrades to a recurring plan.

## Payment Principle

Students may initiate usage, but parents are expected to be the primary payers, especially for students under 18.

The product should support parent payment links and recurring billing.
